Exercício 6 - Box Model Prático

Três caixas demonstrando diferentes comportamentos do box model:

Caixa 1

content-box

width: 200px

padding: 20px

border: 5px

Largura total:
200 + (20×2) + (5×2) = 250px

Caixa 2

border-box

width: 200px

padding: 20px

border: 5px

Largura total:
200px (inclui tudo)

Caixa 3

border-box

width: 50%

padding: 2rem

margin: 1rem

Largura total:
50% do container pai

Cálculo das Larguras:

Caixa 1 (content-box - padrão):

Fórmula: width + padding-left + padding-right + border-left + border-right

Cálculo: 200px + 20px + 20px + 5px + 5px = 250px

O padding e border são adicionados à largura especificada

Caixa 2 (border-box):

Fórmula: width (inclui padding e border)

Cálculo: 200px (já inclui padding e border)

O padding e border são subtraídos da largura especificada

Caixa 3 (border-box com %):

Fórmula: 50% do container pai

Cálculo: 50% de 600px = 300px (aproximado)

O padding de 2rem é incluído nos 50% da largura

Diferenças Principais:

  • content-box: Largura especificada é apenas do conteúdo
  • border-box: Largura especificada inclui conteúdo, padding e border
  • Unidades relativas (%): Proporcional ao elemento pai
  • Unidades fixas (px): Tamanho absoluto
